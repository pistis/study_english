# 영어 사내스터디 하는 내용을 정리하는 웹사이트

## 기술
* express
* mongodb
* reactjs, redux

## setup env
### mongodb
* brew update && brew install mongodb
* sudo mkdir -p /data/db
* sudo chown $USER /data/db

### npm module
* npm install -g webpack webpack-dev-server
* npm install -g babel-cli nodemon cross-env
* npm install
